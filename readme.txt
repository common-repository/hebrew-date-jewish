Plugin for displaying the current Hebrew date.


== Description ==
This plugin displays the Hebrew date using a shortcode.
To use the plugin, upload and activate it.
Then, insert the shortcode [dhds_display_hebrew_date_shortcode] into your content,
and it will display the Hebrew date for the current day.
 
== Installation ==
This section describes how to install the plugin and get it working:
1. Upload 'Jewish_Hebrew_Date.zip' to the /wp-content/plugins/ directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Insert the shortcode [dhds_display_hebrew_date_shortcode] into your content. This will display the Hebrew date for the current day.
